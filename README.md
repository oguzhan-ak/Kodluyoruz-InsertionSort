# Insertion Sort

Kodluyoruz Insertion Sort Projesii

## Aşamalar

[22,27,16,2,18,6]

```
1- [22, 27, 16, 2, 18, 6]
2- [16, 22, 27, 2, 18, 6]
3- [2, 16, 22, 27, 18, 6]
4- [2, 16, 18, 22, 27, 6]
5- [2, 6, 16, 18, 22, 27]
```

## Big-O Gösterimi

O(n^2)

## Time Complexity

Average Case: O(n^2)

Worst Case: Liste azalan şekilde sıralı, Big-O gösterimi : O(n^2)

Best Case: Liste artan şekilde sıralı, Big-O gösterimi : O(n)
